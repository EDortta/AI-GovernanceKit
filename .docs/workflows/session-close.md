# Session Close

At the end of each implementation stage/session:

1. Update `handoff.md` with current status, next steps, blockers/risks, files changed, checks/tests executed, active `work_id`, and date.
2. Update the active epic `RESUME.md`.
3. Ensure `RESUME.md` contains exactly one `Next Step (DO THIS FIRST)`.
4. Add concise lessons to `docs/napkin-lessons.md` when applicable.
5. Review the diff for unrelated changes, generated artifacts, secrets, and scope drift.
